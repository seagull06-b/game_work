박쥐는 키보드의 방향키를 사용하여 이동함.(좌, 우, 위, 아래로 이동 가능) <br>
먹이는 화면의 오른쪽 밖에서 생성됨. <br>
초기에는 목숨을 3개가 주워짐. <br>
박쥐가 먹이와 충돌할 시, 먹이가 화면 밖으로 나갈 시(왼쪽 밖) 목숨이 1 감소 함. <br>
먹이를 10개 맞출때 마다 목숨이 1 증가 함. <br>
게임 실행과 재시작은 스페이스바로 가능함. <br>
